# Software Architecture Playground

Bem-vindo ao meu laboratório de Arquitetura de Software.

Criei este repositório para estudar, documentar e implementar na prática padrões arquiteturais, boas práticas de engenharia de software e tecnologias que utilizo no meu dia a dia.

Mais do que exemplos de código, a ideia é registrar decisões, trade-offs e aprendizados ao longo da evolução de cada projeto.

## Objetivos

- Estudar Arquitetura de Software na prática
- Implementar padrões arquiteturais
- Evoluir minhas habilidades em Engenharia de Software
- Utilizar Inteligência Artificial como aceleradora do desenvolvimento
- Compartilhar conhecimento

## Conteúdo

Em breve este repositório terá exemplos práticos de:

- API Design
- Clean Architecture
- Domain-Driven Design (DDD)
- Event-Driven Architecture
- CQRS
- Saga Pattern
- Outbox Pattern
- Idempotência
- Observabilidade
- Resiliência

## Tecnologias

- TypeScript
- Node.js
- Azure
- PostgreSQL
- Docker
